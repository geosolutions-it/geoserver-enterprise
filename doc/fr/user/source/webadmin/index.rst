Interface Web d'administration 
===============================

L'interface web d'administration est un outil web pour configurer tous les aspects 
de GeoServer, de l'ajout de données au paramètrage des services. Elle est 
accessible via un navigateur web sur http://localhost:8080/geoserver/web dans une 
installation par défaut de GeoServer, bien que cette URL peut varier en fonction 
de l'installation de GeoServer.
