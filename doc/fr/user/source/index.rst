Manuel utilisateur de GeoServer
================================

GeoServer est un serveur logiciel open source écrit en Java qui permet aux 
utilisateurs de partager et d'éditer des données géospatiales. Pensé pour 
permettre l'intéropérabilité, il publie des données à partir des formats majeurs 
de données spatiales en utilisant des standards ouverts.

Ce manuel utilisateur est un guide complet sur tous les aspects d'utilisation de 
GeoServer. Que vous soyez un novice ou un vétéran de ce logiciel, nous espérons 
que cette documentation vous sera d'une grande aide.
